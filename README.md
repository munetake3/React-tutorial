# React-tutorial

[チュートリアル : Reactの導入](https://ja.reactjs.org/tutorial/tutorial.html)をやる

## 環境メモ

- Node.js 16.16.0

## How to Run

```zsh
git clone https://github.com/munetake3/React-tutorial.git
```
or
```zsh
git clone git@github.com:munetake3/React-tutorial.git
```

```zsh
cd React-tutorial
npm install
npm start
```
